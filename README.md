# Kotlin 2D Test Game

This is a project to test Claude Code with Spec based development using openspec.

The goal is to not write any code, but see how far I get by writing openspec specifications and then used Claude Opus to implement them. 

## Gameplay ideas

* Inventory system with weapons and armour.
* Health and magic point system
* Melee damage based on weapon stats and armour class, and dice rolls
* Turn based enemy combat
* Different types of levels with different methods of procedural generation
* Larger pool of items and monsters
